## Critical Rules

| Rule                          | Details                                                                                        |
| ----------------------------- | ---------------------------------------------------------------------------------------------- |
| NO `console.log`              | Only `console.warn` / `console.error` allowed                                                  |
| NO `any`                      | `@typescript-eslint/no-explicit-any: error`                                                    |
| NO relative cross-dir imports | Always use `@/` alias                                                                          |
| Files & folders = kebab-case  | enforced by `eslint-plugin-check-file`                                                         |
| Features isolated             | A feature MUST NOT import from another feature                                                 |
| Shared layers pure            | `components/`, `hooks/`, `lib/`, `types/`, `utils/` MUST NOT import from `features/` or `app/` |
| Boolean naming                | `is*` / `has*` / `should*` / `can*` / `did*` / `will*` / `are*`                                |
| Components ≤150 lines         | Hard warn — split into smaller components                                                      |
| No nested ternary             | Extract to variables / early returns                                                           |
| Strict equality               | `===` / `!==` (null comparison excepted)                                                       |
| Import order                  | builtin → external → internal → parent/sibling → index → object (alphabetized)                 |
| No cycles                     | `import/no-cycle: error`                                                                       |
| Blank line before return      | enforced                                                                                       |
| `routeTree.gen.ts`            | NEVER edit by hand — generated by TanStack Router plugin                                       |

