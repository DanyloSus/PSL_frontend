# React — Storybook

## Overview

| Aspect  | Details                                              |
| ------- | ---------------------------------------------------- |
| Goal    | Visual regression + isolated component development   |
| When    | Every `src/ui/` primitive and any reusable component |
| Version | Storybook 10 with `@storybook/react-vite`            |

## File layout

```
src/ui/button.tsx
src/ui/button.stories.tsx
```

## CSF3 story

```tsx
// src/ui/button.stories.tsx
import type { Meta, StoryObj } from "@storybook/react-vite";

import { Button } from "./button";

const meta = {
  title: "UI/Button",
  component: Button,
  parameters: { layout: "centered" },
  tags: ["autodocs"],
  argTypes: {
    variant: {
      control: "select",
      options: ["default", "ghost", "destructive"]
    },
    size: {
      control: "select",
      options: ["sm", "md", "lg"]
    }
  }
} satisfies Meta<typeof Button>;

export default meta;
type Story = StoryObj<typeof meta>;

export const Default: Story = {
  args: { children: "Click me" }
};

export const Ghost: Story = {
  args: { variant: "ghost", children: "Ghost" }
};

export const Destructive: Story = {
  args: { variant: "destructive", children: "Delete" }
};
```

## Addons installed

| Addon                         | Purpose                         |
| ----------------------------- | ------------------------------- |
| `@storybook/addon-a11y`       | Accessibility violations panel  |
| `@storybook/addon-docs`       | Autodocs + MDX                  |
| `@storybook/addon-vitest`     | Run component tests as stories  |
| `@storybook/addon-onboarding` | First-run tutorial              |
| `@chromatic-com/storybook`    | Visual regression via Chromatic |

## Commands

```bash
npm run storybook        # dev server on port 6006
npm run build:storybook  # build static site
```

## Common mistakes

| Mistake                         | Fix                                  |
| ------------------------------- | ------------------------------------ |
| CSF2 function stories           | Upgrade to CSF3 object form          |
| No `tags: ['autodocs']`         | You lose the autogenerated docs page |
| Missing `argTypes` for variants | Controls won't render                |
| Stories in a separate folder    | Co-locate next to the component      |

## Checklist

- [ ] Story file is `*.stories.tsx` next to component
- [ ] Uses CSF3 (object meta + named exports)
- [ ] `tags: ['autodocs']` set
- [ ] One story per variant
- [ ] a11y panel clean
